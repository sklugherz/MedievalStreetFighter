## MedievalStreetFighter

A simple medieval themed, arcade-style fighing game.

### Controls

- Player 1
  - Left: A
  - Right: D
  - Jump: W
  - Attack 1: R
  - Attack 2: T

- Player2
  - Left: Left Arrow
  - Right: Right Arrow
  - Jump: Up Arrow
  - Attack 1: NUMPAD 1
  - Attack 2: NUMPAD 2

### Whats Implemented:

- Hit Registration
- Basic Game Loop
- In-Game UI
- Character Movement & Attacks
- Starting Menu
